Sistema operativo:
Es el software principal que administra los recursos del computador y permite que los programas funcionen correctamente. También gestiona el hardware, los archivos y la comunicación entre aplicaciones.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza principalmente para desarrollar aplicaciones y servidores web.

Aplicación server.js:
Es el archivo principal que contiene el código del servidor de una aplicación. Su función es recibir solicitudes, procesarlas y enviar respuestas a los usuarios.

Puerto:
Es un número que identifica un canal de comunicación entre el servidor y otros dispositivos o aplicaciones. Permite que los datos lleguen al servicio correcto.

Logs:
Son registros o mensajes que almacenan información sobre las actividades y eventos del sistema o la aplicación. Ayudan a detectar errores y supervisar el funcionamiento del sistema.
